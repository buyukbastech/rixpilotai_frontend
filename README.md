# Apronova AI - Landing Page

Yapay zeka destekli WhatsApp asistanı ve akıllı CRM çözümü.

## Proje Hakkında

Apronova AI, işletmelerin müşteri iletişimini otomatikleştiren, randevuları yöneten ve satış süreçlerini takip eden modern bir platformdur.

### Kullanılan Teknolojiler

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, Framer Motion, Lucide Icons
- **Backend:** Supabase (Database & Auth)
- **UI Components:** Radix UI, shadcn/ui

## Kurulum ve Çalıştırma

1. Bağımlılıkları yükleyin:
   ```sh
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```sh
   npm run dev
   ```

3. Çevre değişkenlerini ayarlayın:
   `.env` dosyası oluşturun ve şu bilgileri ekleyin:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
